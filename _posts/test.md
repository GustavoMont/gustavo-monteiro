# Metadados

## Type

A ideia é que tenha os tipos: `POETRY`, e `TEXT` (talvez adicione outros como `STORY` mas no momento apenas esses).

Esses tipos vão ser úteis tanto para filtros quanto para diferentes estilizações pra cada.

## Tags

As tags vão ser pra categorizar o conteúdo de cada post, podendo criar abas ou separá-los em categorias futuramente.

## Demais campos

Os demais campos são autoexplicativos, e servem para obter dados sobre os posts, fora o próprio conteúdo.

# Texto

Não vou definir nenhuma estrtura de escrita dos posts, caso exista alguma futura vai ser descrita aqui
