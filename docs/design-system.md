# Design System

## Cores

A paleta de cores do projeto será a seguinte:

- Primário: #B131FA
- Secundária: #FF1CC0
- Destaque: #3F8EFC
- Branco: #F1FFFA
- Preto: #272727

[**Visualisação das cores**](https://github.com/GustavoMont/gustavo-monteiro/issues/10#issuecomment-2591307473)

## Estrutura de Texto

A ideia é que cada tipo de poste tenha uma estrutura de texto diferente.

### Tipo: `TEXT`

Para esse tipo de post, inicialmente, será usado a estrutura padrão do [`tailwind-typography`](https://tailwindcss-typography.vercel.app/). Talvez com alguma alteração nas cores do headings

### Tipo: `POETRY`

Ainda não foi definido, mas provavelmente teria que seguir um estilo mais personalizado, e que possa haver mais quebra de linhas etc.

## Tipografia

A única fonte utilizada no projeto é a `Montserrat` e toda variação de tipografia dessa fonte será lidada apenas com o [`tailwind-typography`](https://tailwindcss-typography.vercel.app/)
